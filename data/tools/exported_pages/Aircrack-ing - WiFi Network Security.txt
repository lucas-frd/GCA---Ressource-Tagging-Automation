=Description=
<br>
This open-source tool allows users tomonitor and analyse their own wireless network in order to identify potential security threats. It can be used to crack WEP and WPA-PSK keys, and also allows for the capture of packets in order to carry out further attacks. Aircrack-ng is a powerful tool that should be used with caution, as it can easily be used for malicious purposes.
