=Description=
<br>
SMSPassword is a text message-based password recovery tool. It allows users to recover their password by entering a reset code sent to their mobile phone via text message. The tool is designed to work with any web-based application that uses a text-based password reset system.

How it works:

1. Users enter their mobile phone number into the SMSPassword recovery tool.

2. A text message is sent to the user's mobile phone with a reset code
