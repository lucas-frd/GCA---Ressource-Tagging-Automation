Onlykey is a hardware-based security key that provides two-factor authentication (2FA) and Passwordless logins for online accounts. It works with popular web browsers and apps like Google, Facebook, Dropbox, GitHub, and more. Onlykey is built with a robust security architecture that includes physical security features, tamper-proof hardware, and firmware that has been verified by an independent security audit.
