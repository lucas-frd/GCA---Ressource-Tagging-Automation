Developed by the MITRE Corporation, Xton Access Manager is an open source access control management tool that allows organizations to centrally manage and monitor user access to systems and data. It provides a web-based interface for administrators to manage user accounts, roles, and permissions, and includes a auditing and reporting functionality to track user activity and assess compliance with security policies.
