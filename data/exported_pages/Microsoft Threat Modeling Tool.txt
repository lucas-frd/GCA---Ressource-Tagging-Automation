=Description=
<br>
The Microsoft Threat Modeling Tool helps security practitioners plan and execute threat modeling exercises. The tool walks users through a structured process for identifying, analyzing, and mitigating threats to software systems. The tool can be used to support both design-time and runtime threat modeling.
