=Description=
<br>
The Windows Malicious Software Removal tool is a free Microsoft tool that is used to scan and remove specific prevalent malware from Windows computers. The tool is updated regularly with new definitions for new malware, and can be run manually or scheduled to run automatically. The tool is designed to supplement other anti-malware solutions, and is not a replacement for them.
