The Penetration Testing Framework from the SANS Institute is a collection of tools and documentation designed to help penetration testers and security professionals conduct effective security testing. The tools and documentation are organized into seven categories: network reconnaissance, auditing, privilege escalation, post-exploitation, denial of service, web application attacks, and wireless attacks. The tools and documentation are available for download from the SANS website.
