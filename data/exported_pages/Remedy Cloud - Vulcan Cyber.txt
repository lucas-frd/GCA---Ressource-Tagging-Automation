Vulcan Cyber's Voyager is a tool that can be used to scan for vulnerabilities in cloud-based environments. It uses a combination of static and dynamic analysis to find potential security issues, and then provides a report with information on the severity of each issue and how to fix it. Voyager can be used to scan for vulnerabilities in AWS, Azure, and Google Cloud Platform.
