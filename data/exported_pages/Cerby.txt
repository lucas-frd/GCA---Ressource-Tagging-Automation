Cerby is a cloud-based content management and security platform that helps organizations securely store, share, and collaborate on sensitive information. Cerby provides a central repository for all of an organization's sensitive content, as well as robust security and collaboration features. Cerby's security features include role-based access control, content encryption, and activity logging. Cerby's collaboration features include commenting, tagging, and versioning.
