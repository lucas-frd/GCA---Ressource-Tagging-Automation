The CrowdStrike Reporting Tool (CRT) is a free and open source tool that collects data from the CrowdStrike Falco agent and produces aggregated, JSON-formatted reports. CRT runs on Windows, Linux, and macOS, and can be run from the command line or configured to run as a scheduled task. Reports can be output to a file or sent to a remote syslog server.
